com.cohort.util

Class ScriptMath

  • java.lang.Object
    • com.cohort.util.ScriptMath

  • public class ScriptMath
extends java.lang.Object
    This class makes almost all of the static methods in java.lang.Math accessible to JexlScript scripts as "Math.name()" methods.
